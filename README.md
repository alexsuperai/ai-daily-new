# AI Daily News

每日AI新闻汇总，使用Tavily API收集最新的AI发展、产品和技术突破。

## 项目说明

此存储库包含由自动化脚本每日生成的AI新闻摘要。这些摘要通过Tavily API收集，重点关注：

- 重大AI技术突破
- 新的AI产品发布
- 重要的AI行业动态

## 自动化流程

- 每日上午10:00（上海时间）自动运行
- 收集10个最重要的AI相关新闻
- 生成带日期的Markdown文件
- 自动推送到此仓库

## 文件命名规则

新闻文件按日期命名，例如：
- `daily_ai_news_2026-02-09.md`
- `daily_ai_news_2026-02-10.md`

## 技术细节

- 数据来源：Tavily API
- 更新频率：每日一次
- 内容格式：Markdown
